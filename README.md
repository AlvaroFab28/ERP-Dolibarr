# ERP Dolibarr - Prototipo Funcional

Este repositorio contiene un prototipo funcional de interfaz de usuario para un sistema ERP basado en Dolibarr. El proyecto ha sido desarrollado enfocándose en la experiencia de usuario (UX) y una interfaz moderna (UI), operando enteramente en el lado del cliente (Frontend).

## Tecnologías Utilizadas

*   **HTML5:** Estructura semántica de la aplicación.
*   **CSS3:** Estilos personalizados, diseño responsivo y sistema de diseño visual (sin frameworks pesados).
*   **JavaScript (Vanilla):** Lógica de navegación, renderizado dinámico de módulos y simulación de interacciones de datos.

## Estructura del Proyecto

El prototipo está diseñado de manera modular. La navegación principal permite alternar entre diferentes áreas funcionales simuladas:

*   **Inicio (Dashboard):** Vista general con métricas e indicadores.
*   **Comercial:** Gestión simulada de ventas y clientes.
*   **Financiera:** Resumen de ingresos, gastos y facturación.
*   **Miembros:** Gestión de usuarios y personal.
*   **Sitios:** Administración de locaciones o puntos de venta.

## Instrucciones de Uso

Para visualizar y probar el prototipo, no es necesario instalar ningún servidor local ni dependencias de backend. Siga estos sencillos pasos:

1.  **Descargar el proyecto:** Clone este repositorio o descargue el archivo ZIP y extráigalo en su computadora.
2.  **Abrir la aplicación:** Navegue hasta la carpeta principal del proyecto (por ejemplo, `ERP-Dolibar Proto`) y haga doble clic en el archivo `index.html`. Esto abrirá el sistema en su navegador web predeterminado.
3.  **Credenciales de acceso:** Para ingresar al sistema interactivo, utilice las siguientes credenciales de prueba en la pantalla de inicio de sesión:
    *   **Usuario:** admin
    *   **Contraseña:** admin

## Notas de Desarrollo

Al ser un prototipo frontend, los datos mostrados son generados estáticamente para propósitos de demostración. No existe persistencia de datos en una base de datos real. Cualquier cambio realizado durante la sesión se reiniciará al recargar la página.
